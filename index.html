Last Will - Puzzle Solvers

These puzzle solvers are supposed to help with the randomly generated puzzles in the game Last Will.
They are intended to be used by speedrunners and lazy players on their secondary walkthroughs.

Updates of the game that change the puzzles may always break these solvers.
Currently based on the version from April 3rd 02017.

SPOILER WARNING:
All these solvers will contain gameplay spoilers.
Do not use these solvers on your first encounters.
It will definitely take the fun out of the game.
The game contains sufficient clues to beat it without external help.
Some puzzles have more than one path to the solution. Look around and read all the hints.
Open the ESC menu to access several bonus hints for each room.
If you absolutely do need guidance on your first try, then the Steam guide by LastNonak is recommended.
